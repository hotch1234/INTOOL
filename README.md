# Zen Globe Offline 🌍

A fully offline, zero-dependency interactive globe simulation built with pure HTML5 Canvas and JavaScript.

No CDNs. No libraries. No internet required.  
Just open and run.

---

## ✨ Overview

Zen Globe is a minimalist globe experience that simulates Earth using mathematical rendering instead of WebGL or external APIs.

It focuses on:
- Smooth mobile interaction
- Offline usage
- Lightweight performance
- Zen-like motion experience

---

## 🌍 Features

- 📴 Fully offline (works with no internet)
- 📦 Single-file project (just index.html)
- 📱 Mobile touch + desktop mouse support
- 🌀 Momentum-based rotation physics
- 🌌 Soft atmospheric glow effect
- 🌐 Latitude / longitude globe grid
- ⚡ Ultra lightweight (no dependencies)
