# 🌍 INTOOL — Offline Navigation Engine

INTOOL is a lightweight, fully offline navigation-style web engine inspired by modern map systems.  
It is designed for smooth interaction, zooming, panning, and spatial exploration — all without APIs, CDNs, or external services.

---

## ✨ Overview

INTOOL simulates a global navigation experience using pure HTML5 Canvas and JavaScript.

It focuses on:
- Smooth map-like movement
- Infinite zoom system
- Drag-to-pan navigation
- Lightweight offline performance
- Mobile-friendly interaction

This is not a real-world map service — it is a **self-contained navigation simulation engine**.

---

## 🌍 Features

- 📴 Fully offline (no internet required)
- 📦 Single-file deployment (`index.html`)
- 🧭 Smooth camera-based navigation system
- 🔍 Zoom in / zoom out (Google Maps-style feel)
- ✋ Drag to pan across world space
- 📱 Mobile touch support (swipe navigation)
- ⚡ Lightweight canvas rendering (no libraries)
- 📍 Marker system for points of interest
- 🧠 Inertia motion for natural movement
